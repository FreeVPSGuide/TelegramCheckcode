# 1、阅读必知的4个结论

在解决验证码接收的问题之前，大家一定要搞清楚以下4个结论。

账号触发风控后的拦截提示，一般在旧版本的Telegram（11.5以前的版本），或者第三方版本例如TelegramX、Nicegram是可以绕过这个付费弹窗的。

Telegram X收不到验证码，核心原因是网络风控、短信拦截或操作触发平台限制等3种因素导致的。

**1、添加「搜索机器人」**

新注册的电报号，打开TelegramX是一偏空白的，所以必须添加一些「搜索机器人」。

![](https://files.mdnice.com/user/4562/9a58e12a-c319-41fd-bbfb-5977a501a0c3.jpg)

复制以下任意链接到浏览器，通过浏览器即可点击启动Telegram，然后点击「开始」即可加入。

![](https://files.mdnice.com/user/4562/a7de9255-b9cf-45c3-bb3b-c52c71a5686a.jpg)
```
极搜：
https://t.me/jisou?start=a_8522386636

快搜：
https://t.me/kuaiaa?start=a_3WY0BP7

新币搜：
https://t.me/xbso1?start=a_8522386636
```

最好是把这3个搜索机器人都添加上，每个「搜索机器人」是的信息来源是不一样的，这个搜不到就换一个，强烈推荐多个备用。

国内的微信能搜索到你「加入」的微信群「聊天记录」，但绝对不可能搜索到你「未加入」的微信群的「聊天记录」。

几乎每个Telegram搜索机器人都有以下这3个特色功能：

1、能查看每天的热搜榜单

2、在菜单中能直接找到已经「分类」好的群组。

3、能搜索到各种好玩的机器人、群组、频道（相当于公众号）、资源（电影、图片、新闻、音乐、电子书）

**2、添加邮箱地址**

必须添加邮箱地址，不然到时候没有手机号接收验证码，绑定了邮箱地址，就可以使用俄版的Telega在邮箱中接收验证码。

TelegramX没有看到绑定邮箱地址的这个功能。

Telegram在「设置」>「隐私安全」>「邮箱登录」中可以设置邮箱。

**6、隐藏手机号(保护隐私)**


操作路径：设置(Settings)〉隐私与安全(Privacy and Security)〉手机号码(Phone Number)。

修改建议：将"谁可以看到我的手机号码"设置为"没有人(Nobody)"。

这样可以有效防止别人通过软件看到你的真实手机号。
